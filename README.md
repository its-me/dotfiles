## hotkeys

### niri

| hotkey | action |
| :--- | :--- |
| Super+Shift+/ | show hotkeys panel |
| Mod+Shift+R | switch window height |
| Mod+[ | move window to column on left |
| Mod+] | move window to column on right |

### yazi

| hotkey | action |
| :--- | :--- |
| Ctrl+Z | toggle block/status bar visibility |

### zsh

| hotkey | action |
| :--- | :--- |
| Ctrl+R | run hstr |
